# myskills

Open-source Codex skills for small, durable workflows.

## Skills

- `notetaker`: writes and maintains durable session notes so future Codex sessions can preserve what was discussed, decided, changed, completed, why it happened, and what should happen next.

## Use

Install or copy the skill folder into your Codex skills directory, then invoke it as `$notetaker`.

```bash
cp -R notetaker "${CODEX_HOME:-$HOME/.codex}/skills/"
```
